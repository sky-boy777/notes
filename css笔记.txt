清除浮动：
原因：父盒子不方便给高度的情况下，子元素设置浮动会撑不开高度，使父盒子的高度变为0，影响后面的盒子。
1、父级添加overflow属性
2、父级添加alter伪元素
3、父级添加双伪元素
4、了解：额外标签，在最后一个浮动的元素添加一个块元素，添加clear：both；

父级元素添加：overflow：hidden；   缺点溢出隐藏

父级元素添加伪元素：
.clearfix: after {         添加伪元素，额外标签法升级版
    content: "";
    display: block;
    height: 0;
    clear: both;
    visbility: hidden;
}
.clerfix { *zoom: 1; }     兼容ie67
优点：没有增加标签，结构更简单
缺点：照顾低版本浏览器

父级添加双伪元素清除浮动：
.clearfix:before, .clearfix: after {          前后添加伪元素
    content: "";
    display: table;
}
.clearfix: after { clear: both; }
.clerfix { *zoom: 1; }     兼容ie67
